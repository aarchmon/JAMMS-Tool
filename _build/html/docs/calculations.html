calculations module¶

Calculations Module¶

Financial calculations module which acts upon a Pandas DataFrame object containing information about assets. A set of various computations for this data is defined here.

NOTE: All functions within this module will operate under the assumption that the DataFrame instance indicates closing

prices of assets.

calculations.calculate_average_annual_returns(daily_returns_df)¶

Calculates the annual returns on daily returns. Must only be ran after calculate_daily_returns() method has been called.

Parameters

daily_returns_df (Pandas DataFrame.) – Daily returns.

calculations.calculate_average_annual_volatility(daily_returns_df)¶

Calculates the annual volatility on daily returns. Must only be ran after calculate_daily_returns() method has been called.

Parameters

daily_returns_df (Pandas DataFrame.) – Daily returns.

calculations.calculate_daily_returns(closing_prices_df)¶

Calculates the daily returns on closing prices.

Parameters

df (Pandas DataFrame.) – Closing prices.

calculations.calculate_portfolio_return(daily_returns_df, weight)¶

Calculate the portfolio return per a given weight based off of client risk profile.

Parameters

  • average_annual_returns_df (Pandas DataFrame.) – Average annual returns of asset data.

  • weight (float list) – Portfolio weight per client risk profile.

calculations.calculate_portfolio_sharpe_ratio(portfolio_return_df, portfolio_volatility)¶

Calculate the sharpe ratio of the portfolio. Must only be ran after calculate_portfolio_return() and calculate_portfolio_volatility().

Parameters

  • portfolio_return_df (Pandas DataFrame.) – Average annual returns of the portfolio.

  • portfolio_volatility (float) – Portfolio volatility

calculations.calculate_portfolio_volatility(daily_returns_df, weight)¶

Calculate the volatility of the portfolio. Must only be ran after calculate_daily_returns() and risk_profile() methods have been called.

Parameters

  • daily_returns_df (Pandas DataFrame.) – Daily returns.

  • risk_prof (string) – client risk profile.

calculations.calculate_sharpe_ratio(avg_annual_returns_df, avg_annual_volatility_df)¶

Calculates the Sharpe Ratio of a given portfolio. Must only be ran after calculate_daily_returns(), calculate_average_annual_returns() calculate_average_annual_volatility() methods have been called.

Parameters

  • avg_annual_returns_df (Pandas DataFrame.) – Average annual returns.

  • avg_annual_volatility_df (Pandas DataFrame.) – Average annual volatility.
